Пользователь вводит строку. Если первый символ строки — буква "А", выведите: "Строка '[строка]' начинается с буквы А.", иначе — "Строка '[строка]' не начинается с буквы А.". Решение через: if ... else, интерполяцию. Входные: "Авокадо" → Результат: "Строка 'Авокадо' начинается с буквы А." Входные: "Банан" → Результат: "Строка 'Банан' не начинается с буквы А."
