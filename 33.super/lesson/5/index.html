Создайте базовый класс Shape с методом getArea(), который возвращает площадь фигуры. Реализуйте подклассы Rectangle, Circle и Triangle, которые наследуют от Shape и реализуют метод getArea() для расчета площади по формулам: • Прямоугольник: area = width * height • Круг: area = π * radius^2 • Треугольник: area = (base * height) / 2
