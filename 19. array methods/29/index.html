29. На входе n – количество элементов массива, затем пользователь заполняет массив с клавиатуры. Использовать метод filter, чтобы создать новый массив, содержащий только элементы, которые являются положительными числами. Входные: [1, -2, 3, -4, 5] → Результат: [1, 3, 5] Входные: [0, -10, -20] → Результат: "Массив пуст" Входные: [10, 20, 30] → Результат: [10, 20, 30]
